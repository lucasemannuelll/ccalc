# ccalc

A simple C-based calculator for basic arithmetic operations.

## Features

- Addition, subtraction, multiplication, and division.
- Basic error handling (division by zero, invalid operators).
- Built with `clang` and managed with `just`.

## Building

To build the project, ensure you have `just` and `clang` installed, then run:

```bash
just build
```

## Usage

Run the compiled binary:

```bash
./ccalc
```

Then follow the prompts to enter two numbers and an operator.
